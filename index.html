Persiapan

Pastikan semua tools sudah terinstall sebelum memulai

!

Untuk instalasi Laravel, kamu hanya butuh XAMPP (sudah include PHP & MySQL) dan Composer. Pastikan keduanya sudah terinstall di komputer kamu.

XAMPP

Sudah include PHP & MySQL

Download XAMPP →

Composer

Package manager untuk PHP

Download Composer →
Tips

XAMPP sudah include PHP di dalamnya, jadi kamu tidak perlu install PHP secara terpisah. Cukup install XAMPP, lalu Composer, dan kamu siap lanjut ke langkah berikutnya!

Cek Versi PHP

Buka terminal / command prompt, lalu ketik perintah berikut untuk memastikan PHP sudah terinstall:

Terminal 
php -v
Output yang diharapkan: 
PHP 8.2.0 (cli) (built: ...)
Copyright (c) The PHP Group

Cek Versi Composer

Terminal 
composer -v
02

Aktifkan Extension ZIP di PHP

Wajib dilakukan sebelum instalasi Laravel — khusus pengguna XAMPP

!

Laravel membutuhkan extension zip di PHP. Di XAMPP, extension ini biasanya dinonaktifkan secara default. Kita perlu mengaktifkannya dulu melalui file php.ini.

① Buka Folder PHP di XAMPP

Pergi ke folder instalasi XAMPP di komputer kamu, lalu masuk ke folder php. Biasanya ada di lokasi berikut:

Lokasi XAMPP (Windows) 
C:\xampp\php\
1
Buka File Explorer
→
2
Masuk ke C:\xampp\php
→
3
Cari file php.ini

② Buka php.ini dengan Visual Studio Code

Klik kanan pada file php.ini, lalu pilih "Open with" → pilih Visual Studio Code.

1
Klik kanan php.ini
→
2
Klik Open with
→
3
Pilih Visual Studio Code
Tips — Cara Cepat Cari Teks di VS Code

Setelah file terbuka, tekan Ctrl + F untuk membuka kotak pencarian.

③ Cari Baris ;extension=zip

Tekan Ctrl + F untuk membuka pencarian di VS Code, lalu ketik:

Ctrl + F — Cari di VS Code 
;extension=zip

Tanda titik koma (;) di awal baris artinya baris tersebut dinonaktifkan / di-comment. Kita perlu menghapus tanda itu agar extension aktif.

④ Hapus Tanda Titik Koma

Ubah baris tersebut dari ini:

Sebelum (nonaktif) 
;extension=zip

Menjadi seperti ini (hapus titik koma di depannya):

Sesudah (aktif) 
extension=zip

⑤ Simpan File

Tekan Ctrl + S untuk menyimpan perubahan. Selesai! Extension ZIP sekarang sudah aktif.

03

Buat Project Laravel

Buat project Laravel baru di dalam folder htdocs XAMPP

① Buka Terminal / CMD

Buka Command Prompt (tekan Windows + R, ketik cmd, tekan Enter). Lalu arahkan ke folder htdocs milik XAMPP dengan perintah berikut:

CMD / Terminal 
cd \xampp\htdocs
📁

Folder htdocs adalah tempat menyimpan semua project web di XAMPP. Laravel harus dibuat di dalam folder ini supaya bisa diakses lewat browser.

② Buat Project Laravel Baru

Setelah berada di folder htdocs, jalankan perintah berikut untuk membuat project Laravel. Ganti nama-project dengan nama yang kamu inginkan:

CMD / Terminal 
composer create-project laravel/laravel nama-project

Penjelasan Perintah

  • composer — memanggil Composer
  • create-project — membuat project baru
  • laravel/laravel — package Laravel yang akan diinstall
  • nama-project — nama folder project kamu, misal: toko-online
Contoh lengkap 
cd \xampp\htdocs
composer create-project laravel/laravel toko-online
⏳

Proses download dan install memerlukan koneksi internet dan bisa memakan waktu beberapa menit. Kamu akan melihat banyak teks berjalan di CMD — itu normal, tunggu sampai selesai!

04

Masuk ke Folder Project

Arahkan CMD ke dalam folder project yang baru dibuat

Setelah instalasi selesai, masuk ke dalam folder project kamu dengan perintah cd diikuti nama project yang tadi kamu buat:

CMD / Terminal 
cd nama-project

Contoh

  • Jika tadi kamu buat project bernama toko-online, maka ketik: cd toko-online
  • Setelah enter, kamu sekarang berada di dalam folder project Laravel kamu
Contoh nyata 
cd toko-online
Tips

Kamu bisa cek posisi kamu sekarang dengan melihat path di CMD. Seharusnya tertulis seperti: C:\xampp\htdocs\toko-online>

05

Jalankan Laravel

Satu perintah, dan Laravel langsung berjalan!

Pastikan kamu sudah berada di dalam folder project (langkah 4). Sekarang ketik perintah berikut di CMD dan tekan Enter:

CMD / Terminal 
php artisan serve

Di CMD akan muncul pesan seperti ini:

Output di CMD: 
   INFO  Server running on [http://127.0.0.1:8000].

  Press Ctrl+C to stop the server
🖱️

Tekan Ctrl + klik pada link http://127.0.0.1:8000 di CMD untuk langsung membuka browser. Atau ketik manual di browser kamu.

Laravel berhasil berjalan!

Buka browser dan akses: http://127.0.0.1:8000

Tampilan yang akan muncul di browser:

127.0.0.1:8000
! Cara Menghentikan Server

Untuk menghentikan server Laravel, tekan Ctrl + C di CMD. Jika CMD ditutup, server otomatis berhenti.
